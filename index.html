Have you ever wondered why it's so hard to read anything on your mobile device while commuting to work or friends? The bus or taxi driver simply annoys you because he/she really wants to beat his/her personal record?

Fear no more. That why I want to make unwobble, a mobile solution that uses your mobile's accelerometer to balance the screen's content so that it becomes much more easier to read.

See FAQ below to learn more.

But why this site? Well, to make this project happen I need a bit of funding to pay the rent and getting some coffee for a few months to work on unwobble.

Interested? Become a backer of unwobble by clicking on "I'll support it!" above. This will help me getting the project off the ground and working towards a first prototype. If this all is gonna work? I don't know to be honest, but it really sounds like a great idea worth chasing after.

Further plans include: Location awareness for letting unwobble know if it needs to adjust its algorithms on heavily shaking roads, support for the entire operating system to make unwobble available to third-party applications (only for rooted Cyanogenmod-based phones as a start).

FAQ

What exactly is unwobble?
unwobble is a new way of reading and displaying content while commuting. It provides screen content stabilization for mobile devices to compensate shaking using the built-in accelerometer.
With which mobile devices will unwobble work with?
For a first version only Android-based phones are supported. An iPhone and Windows Phone application might come in the future.
What's about Symbian-based phones or Windows Phone devices?
Symbian will not be supported, a Windows Phone version is planned.
What will unwobble cost?

I didn't make a decision about pricing yet. However, there definitely will be a demo version which you can give a try before buying. Development of such a project will take up a lot of my time.

Purchasing certainly won't burn a hole into your wallet -- it'll cost less then you'd normally pay for a good cup of coffee.

Are there any special requirements my phone needs to meet?
Yup, besides having a supported phone (see question above) an accelerometer needs to be built in. Don't worry if you're not sure you have one. Just give unwobble a try as soon as it's released.
